# AI職場外掛 · 社群介紹頁

> 我用AI解決了很多工作上的麻煩事，這裡是我的紀錄，也希望對你有用。

## 關於這個專案

這是「AI職場外掛」的品牌介紹網站，用 React + Vite 製作，部署在 Vercel。

內容涵蓋：
- 我的故事與創作理念
- 我在做什麼
- 內容主題（職場AI技巧、Prompt、工具評測）
- 各平台社群連結

## 技術架構

- **Framework**: React 18
- **Build Tool**: Vite
- **Deploy**: Vercel
- **字型**: Noto Sans TC

## 本地開發

```bash
npm install
npm run dev
```

## 部署

推送到 GitHub 後，Vercel 會自動偵測並重新部署。
